Calendar

This is a calendar for you adding events. You could click on any date which belong to this month to add an event. If you add more than three events on the same date, it will warn you. If you continue to add events, a scroll will show in the block of the date. When you put your mouse on the block of each date, there will be details you put in. If you add an wrong message, you can click it and delete it. When you complished the event, you can set its status to finished by clicking on it. If you want to print it out, you should add the event to the plan list on your right, and then click on the print button to print.
